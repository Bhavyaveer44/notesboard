ThinkBoard is a modern note-taking application that allows users to:

Create, read, update, and delete notes </br>
View all notes in a responsive grid layout </br>
Edit note details with real-time feedback </br>
Experience rate limiting to prevent abuse


Tech Stack:-

Backend- </br>
Runtime: Node.js </br>
Framework: Express.js </br>
Database: MongoDB (with Mongoose ODM) </br>
Rate Limiting: Upstash Redis </br>

Frontend- </br>
Library: React 19 </br>
Bundler: Vite </br>
Routing: React Router v7  </br>
HTTP Client: Axios </br>
Styling: Tailwind CSS + DaisyUI </br>
UI Icons: Lucide React </br>
Notifications: React Hot Toast </br>
